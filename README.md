# 🏛️ ArchMind — Software Architecture Mastery Platform

**ArchMind** is a premium, interactive learning platform designed for modern software architects. It bridges the gap between traditional **Enterprise .NET Architecture** and the cutting-edge world of **AI Engineering**.

![Tech](https://img.shields.io/badge/Stack-.NET%20%7C%20AI%20%7C%20JS-blue?style=for-the-badge)

---

## 🌟 Key Features

### 💎 1. The 3D Architecture Hub
A custom-built, high-fidelity 3D engine that visualizes architectural concepts as interactive neural meshes. Explore the "shape" of your code in real-time.

### 🧩 2. .NET Architect Track
A deep-dive curriculum covering foundational to advanced patterns:
- **ASP.NET MVC Masterclass**: Understanding the separation of concerns.
- **OOP & SOLID**: Master the principles of clean, maintainable code.
- **GoF Design Patterns**: Interactive catalog of all 23 Gang of Four patterns with production-ready C# examples.

### 🌐 3. Microservices & Distributed Systems
A professional-grade guide to building scalable systems:
- **Master Topology**: Understanding Gateways, Service Discovery, and Event Buses.
- **Advanced Patterns**: Deep dives into **SAGA (Choreography/Orchestration)**, **CQRS**, **Event Sourcing**, and **Circuit Breakers (Polly)**.
- **Visual Blueprints**: High-fidelity SVG diagrams and professional case studies.

### 🤖 4. AI Engineering Track
Future-proof your career with advanced LLM architecture knowledge:
- **How LLMs Work**: Demystifying Transformers, Attention Mechanisms, and Tokenization.
- **RAG Systems**: Architecture for Retrieval-Augmented Generation, Semantic Search, and Vector Databases.

## 🛠️ Technology Stack

- **Core**: Vanilla HTML5, CSS3 (Modern Glassmorphism Design), and Modular JavaScript.
- **Engine**: Custom 2D/3D Canvas engine for high-performance visualizations.
- **Syntax**: [Highlight.js](https://highlightjs.org/) for crystal-clear code snippets.
- **Deployment**: Optimized for **Vercel** and **GitHub Pages**.

## 🚀 Getting Started

### Local Development
The project is built with zero dependencies for maximum speed and simplicity. 
1. Clone the repository:
   ```bash
   git clone https://github.com/sidehustlesCoder/ArchMind.git
   ```
2. Open `index.html` in any modern browser.
3. (Recommended) Run via a local server (e.g., Live Server in VS Code) for the best experience.

### Deployment
The project includes a `package.json` for easy deployment to **Vercel**:
```bash
npm i -g vercel
vercel
```
